# My Routine (Web App)

## Introdução
O My routine é uma aplicação web focada em auxiliar o gerenciamento da rotina diária do usuário com funcionalidades como: Lista de tarefas, painel de avisos, caderno de anotações e outros.

The My Routine is an web-application focused on helping the user's diary routine managment with features like: Task list, notice board, notebook and others. 

O My Routine é ideia de projeto que tive recentemente para fortalecer meus conhecimentos em react e adicionar alguns também. Além disso, esse projeto será realizado para suprir uma necessidade de organização que tenho atualmente, e creio que muitos tenham também (Sei que existem muitos apps mobile e web de lista de tarefas, mas muitos não têm as funcionalidades que quero, e se têm são pagas). Por mais que um site de lista de tarefas seja um dos projetos mais recomendados para os iniciantes(o que torna o projeto meio que muito básico), eu decidi fazê-lo adicionando funcionalidades interessantes.

### Funcionalidades

Visando obter um panorama de tudo o que quero fazer, criei uma lista que é classificada com números de 1 a 5. Os itens próximos do número 1 ou iguais a 1, são os mais importantes, e que consequentemente devem ter um foco mais imediato. Os itens com número mais próximo de 5 ou igual a 5, são ideias que não precisam ser feitas agora (ou que não serão feitas). 

Aqui está a lista:

- Lista de Tarefas em cache. [1]
- Bloco com streak. [2]
- Tabela de avisos. [4]
- Matriz de Eisenhower. [5]
- Histórico de missões concluidas. [4]
- Visão geral do avanço. [5]
- Visão geral detalhada com resumos e etc... com uma IA. [5]
- Deploy com vercel. [5]
- Animações e coisas do tipo. [5]
- Tarefas compartilhadas. [5]
- Popup com informações sobre a tarefa clicada. [2]
- Aviso sobre Beber Água [3]
- Caderno de anotações [3]
- Cadastro de usuários e etc.. [5]

Atualmente quero que o projeto funcione apenas no notebook ou celular do usúario, sem login e conexão com banco de dados para ter acesso em outros dispositivos.
